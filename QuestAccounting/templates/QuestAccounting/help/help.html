Help Section

Homepage

The homepage displays six important ratios every company needs to consider. These ratios are displayed in red if the ratio is considered bad, yellow for ratios that are considered ok, and green for ratios that are considered great.

User View

This page displays the list of users and allows the viewing of their account information.

User Creation

This page is an admin only page that allows admin to create accounts for users.

Notifications

This page displays notifications for managers and admins.

Accounts

This page allows users to see the chart of accounts, add accounts, edit accounts, deactivate accounts, and see each accounts general ledger.

Event Logs

This page displays changes made to account information.

Journal Entries

This page allows users to see the list of approved journal entries, pending journal entries, and a list of all journal entries. Users can add documents to journal entries through the view journal entries tab. This page also allows the adding of journal entries for managers, and the request for accountant users.

Financial Sheets

This page displays the trial balance, balance sheet, income statement, and retained earnings statement. Users can print and save these documents from this section.

Email User

This page allows the user to email an user by selecting an existing account.

User Account

This page displays the users information and allows them to edit their own information.
